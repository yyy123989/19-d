# USART1 AD7606B Wave Output Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Configure USART1 on PA9/PA10 and periodically print AD7606B 8-channel voltage samples to a PC serial tool.

**Architecture:** Keep AD7606B sampling at 40 kSPS in TIM3. Add a low-rate serial output path in the main loop that copies the latest sample and sends one CSV line around every 10 ms. Use a lightweight USART1 register init at 115200 8N1 because this project does not include HAL UART driver files.

**Tech Stack:** STM32F407VET6, USART1 PA9/PA10, Keil ARMCC 5, AD7606B signed-code to millivolt mapping.
